# Unemployment Analysis with Python
## CodeAlpha Data Science Internship — Task 2
Author: Chandra Hasini

#About
Analyzed India's unemployment data (2019-2020) using Python.
explored the impact of COVID-19 on unemployment rates across 28 states.

## Libraries Used
- Pandas
- Matplotlib
- Seaborn

## Steps Covered
1. Analyzed unemployment rate data
2. Data cleaning, exploration & visualization
3. evaluated COVID-19 impact
4. Identified key patterns & seasonal trends
5. Presented policy insights

## Key Findings
- COVID increased unemployment from 9.5% to 17.7%
- Peak month was May 2020 (24.88%)
- Tripura was the most affected state (28.35%)
- Urban areas were hit harder than Rural areas
